This chapter tells you everything needed for scripting in Saga.(For revision: 593)

 

scripts are stored under scripts\npc folder

 

Saga uses c# as scripting language,supports abstract classes(they will not be loaded) and inheritance.

 

All scripts are inherited from Npc class or MapItem/MailBox/Ship class

 

We strongly recommend you to script in Visual Studio .Net 2005 and refers SagaDB.dll Sagamap.exe
